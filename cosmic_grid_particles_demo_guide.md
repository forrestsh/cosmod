# FCC 粒子动力学演示说明

> Cosmic Grid 框架下，FCC 12 方向格点上五种粒子物理事件的可视化。
> 每个场景都是一段精心编排的"剧本"，有具体的时间戳和坐标。

---

## 通用设计

### 12 个方向

每个 cell 的状态是 12 位二进制向量。每位代表一个方向上是否携带信号。

| 族 | 方向 | 颜色 | 含义 |
|---|---|---|---|
| **A 族**（gun 产生） | A, B, C | 琥珀 → 橙 → 红 | rt_FCC 主 cycle |
| **前向 4-cycle** | D, E, F | 翠绿 → 蓝绿 → 青 | 物质 quark |
| **反 A 族** | a, b, c | 淡琥珀 → 淡橙 → 淡红 | 反 gun |
| **后向 4-cycle** | d, e, f | 淡翠绿 → 淡蓝绿 → 淡青 | 反 quark |

球的颜色 = 该 cell 上所有 active 方向颜色的平均混色；popcount 越高球越大（pc≥4 的复合粒子放大 1.4×）。

### 三种规则

- **rt_FCC**：仅 A-cycle，最简单的图灵完备规则
- **rt_P1**：A-cycle + 前/后 4-cycle + Aa↔Bb 单 swap（首次实现对湮灭）
- **rt_6cycle**：rt_P1 把单 swap 换成完整色轮 Aa→Bb→Cc→Dd→Ee→Ff→Aa

### 控制

- 单指拖拽：旋转视角
- 双指捏合：缩放
- 滑动条：跳到任意 tick
- ▶ ⏸：自动播放（每 700ms 一帧）
- ◀ ▶ 单步前后
- ⟲ 重置到 t=0
- **TRAILS**：开关粒子轨迹（前 6 帧的位置淡白点）
- **GRID**：开关 FCC 子格点的"幽灵"显示

底部琥珀色字幕在关键 tick 自动出现；其他时候显示场景描述。

## 场景 1：BDEF 重子（rt_P1）

**这是 4-bit 束缚态。** 在 rt_P1 下，B/D/E/F 四个方向构成一个前向 4-cycle：B→D→E→F→B。一个 cell 同时持有这四个 bit 时，它们以同步周期 4 运行——形成 Cosmic Grid 里最简单的"重子"。

### 动力学

| t | 状态 | 描述 |
|---|---|---|
| 0 | 1 cell, 4 bits 在原点 | **紧凑态** |
| 1 | 4 cells, 各 1 bit | 散开（B,D,E,F 各飞一格） |
| 2 | 4 cells, 各 1 bit | 4 个 bit 在 cycle 里嬗变 |
| 3 | 4 cells, 各 1 bit | 即将重新汇聚 |
| 4 | 1 cell at (1,0,1) | **回归紧凑态**！漂移完成 |
| 8 | 1 cell at (2,0,2) | 再次紧凑 |

### 关键观察

- **周期 4 心跳**：每 4 ticks 回到紧凑态，平均速度 √2/4 ≈ 0.354c（c = √2 是 FCC 光速）
- 中间 3 个 tick 下复合粒子"展开"成 4 个分离的 quark，但它们的几何路径精确对位让它们必然重逢
- 这是**几何束缚**，不是力——T 算子的几何强制实现"自动归位"

## 场景 2：B + b 对湮灭（rt_P1）

**最小的对湮灭事件。** 一个 B 粒子（前向 quark）和一个 b 粒子（反 quark）头对头相向，撞击后湮灭转化为 A+a，A 启动永续 gun，a 作为反粒子辐射飞远。

### 动力学

| t | 事件 | 物理诠释 |
|---|---|---|
| 0 | B 在 (-1,0,-1)，b 在 (1,0,1) | 粒子-反粒子对相向 |
| 1 | 两者在 (0,0,0) 相遇 → rt[B+b] = A+a | **湮灭！**rt[130]=65 触发 |
| 2 | A 飞到 (0,1,1)；a 飞到 (0,-1,-1) | A 和 a 沿相反方向分离 |
| 3 | A bit 单独到达 → rt[A] = A+B → **gun 点燃** | 永续信号流启动 |
| 4+ | gun 头沿 (0,t,t) 推进，B 尾巴脱落（rt[B]=D 在 P1 下） | 标准 gun 行为 |

### 关键观察

- **能量重组**：2 个静态质量 → 1 个 gun（无限能量输出）+ 1 条反粒子辐射
- 整个反应**只用 1 个 swap rule** rt[130]↔rt[65] 实现——但前提是 A-ring 已存在，让 A bit 可以触发 gun
- 类比 e⁺e⁻ → 2γ，但 FCC 的输出更复杂：能量直接以"持续信号生产"形式释放

## 场景 3：重子-反重子对撞（rt_6cycle）⭐

**完整的 30-tick 物理剧本。** BDEF 重子从 (-4,0,-4) 漂移而来，bdef 反重子从 (4,0,4) 漂移过来。它们在 t=12 接近，在 t=13-20 之间发生**多阶段对撞反应**——这是 Cosmic Grid 框架内第一个完整的复合-复合反应，**完全自发**，不需要任何额外规则。

### 完整剧本

| t | 事件 | 物理诠释 |
|---|---|---|
| 0 | 两重子分别在 (-4,0,-4) 和 (4,0,4)，紧凑态 | 初始条件 |
| 4, 8, 12 | 各自周期性回归紧凑态，逐步靠近 | 每 4 ticks 漂移 (1,0,1) |
| 12 | 重子在 (-1,0,-1)，反重子在 (1,0,1) | **即将相遇** |
| 13 | 原点出现 **{C,c}** | **Compton 辐射**：色轮触发 rt[260]=520 |
| 14-15 | C 和 c 沿 ±(1,1,0) 方向各自飞远；其余 6 bits 散落 | Compton wires 离开战场 |
| 16 | 6 bits 重聚于原点形成 **{B,D,E,b,d,e}** | **碰撞核**！pc=6 复合态 |
| 17-18 | 内部嬗变继续 | 6-cycle 继续作用 |
| 19 | 原点变为 **{A,a}** | **湮灭事件**！rt[Aa]=Bb |
| 20 | A 飞到 (0,1,1) → **gun 点燃**；a 自由 wire 飞向 -(0,1,1) | gun 启动 |
| 20+ | gun 持续输出信号，C/c Compton wires 已飞到 (±7,±7,0) | 永续辐射 |

### 三种粒子的并行演化

到 t=20 时，原始 8 个 bit 被重组为：
- **2 条 Compton wires**（C, c）— 飞向 ±(1,1,0) 方向
- **1 条反粒子辐射**（a）— 飞向 -(0,1,1) 方向
- **1 把 gun**（A 出发）— 沿 +(0,1,1) 永续生产新信号

### 关键观察

- **多阶段对撞**：单个对撞事件分 3 个阶段——辐射（t=13）、碰撞核（t=16）、湮灭（t=19）。这在标准模型对撞中也常见。
- **辐射 + 湮灭 + 信号生产**同时发生，对应能量守恒、电荷守恒、伴随辐射。
- 报告里写得很明白：**"无需任何额外规则就自发产生"**。整套机制来自 6-cycle 基础规则 + 两粒子周期-4 轨道的几何相遇。这是 FCC 规则空间内"类物理行为天然涌现"的最强证据。

## 场景 4：Light-speed gun（rt_FCC）

**最经典的 rt_FCC light-speed gun**——单 A bit 种子产生 slope=1 的永续信号流。这是 Cosmic Grid 框架内"图灵完备"的最简实现，作为前面复杂场景的对照基线。

### 动力学

| t | 状态 | 描述 |
|---|---|---|
| 0 | A 在 (0,0,0) | 种子 |
| 1 | A+B 在 (0,1,1) | rt[A]=A+B（cycle 启动） |
| 2 | A+B 在 (0,2,2)；B 在 (1,1,2) | gun 头推进，B 尾巴脱落 |
| t | A+B 在 (0,t,t)；B-trail 沿对角线 | bits(t) = t+1（线性） |

### 关键观察

- **单 cycle = 单 gun head**：信号每 tick 增加 1 bit
- **方向偏向极强**：所有信号都集中在 (0,+y,+z) 走廊里，因为 A、B、C 三个方向都在 (+,+,+) 八分体内
- 与场景 1-3 的复杂物理对比，这个场景显示"最简图灵完备规则"的极简——只有一条 cycle，但它已足以支撑任何可计算函数

## 场景 5：色轮 F+f → gun（rt_6cycle）

**演示 6-cycle 的色轮性质。** F+f 头对头相遇后，rt 触发**色轮闭环**rt[Ff]=Aa——5 个嬗变通道走完一圈，最终回到 A 族。A 触发 gun。

### 动力学

| t | 事件 | 色轮位置 |
|---|---|---|
| 0 | F 在 (-1,0,-1)，f 在 (1,0,1) | 起点 |
| 1 | F+f 在 (0,0,0) → rt[Ff] = **Aa** | **色轮闭环**：Ff → Aa |
| 2 | A 飞到 (0,1,1) → gun 启动；a 飞向 -(0,1,1) | gun 点燃 |

### 关键观察

- **色轮的 6 个通道**全部 pc=2 swap：Aa↔Bb↔Cc↔Dd↔Ee↔Ff↔Aa（环形）
- 5 个通道是**嬗变**（粒子换种，无能量释放），仅 F+f → A+a 这一个通道触发 gun（因为只有 A 是 gun 入口）
- 类比类标准模型中"大部分反应守恒粒子数，仅少数特殊反应触发级联"
- 6 个通道也是**对偶的**：色轮另一方向上 Aa → Ff，所以 A+a 也能转化回 F+f；动力学是可逆的

## 物理总结

可视化的五个场景共同展示了 Cosmic Grid 框架的几个核心论断：

1. **物质谱由 T 算子限定**：所有束缚复合粒子（场景 1, 3）必须以统一速度漂移；BDEF 重子是最大尺寸（4 bits），因为前向 4-cycle 只有 4 相
2. **反应谱由 R 决定**：从场景 2 的单 swap，到场景 5 的 6-cycle 色轮，反应丰富度可调
3. **类物理行为自发涌现**：场景 3 的多阶段对撞**完全是基础规则下的自发动力学**，没有额外规则——这暗示了 Cosmic Grid 规则空间内可能天然涌现"类粒子物理"行为
4. **图灵完备性结构稳健**：所有这些规则变体都保持图灵完备（gun + NOT + AND 通过测试），证明计算性与物理性是代数独立的两个维度
